# SMM PRO — Лендинг курса по SMM

Демонстрационный одностраничный лендинг для онлайн-курса по SMM-продвижению.

**[🔗 Посмотреть сайт](https://asyalapa.github.io/smm-pro-landing/)**

---

## 🧠 О проекте

Это **демонстрационный проект**, созданный для портфолио веб-разработчика.  
Цель — показать навыки вёрстки, стилизации и программирования на vanilla-стеке.

### 🛠 Стек

- **HTML5** — семантическая вёрстка, accessibility (ARIA, roles)
- **CSS3** — кастомные свойства, БЭМ, glassmorphism, адаптив, анимации
- **JavaScript (ES6+)** — 11 IIFE-модулей, IntersectionObserver, маска телефона
- **Material Symbols** — иконки через Google Fonts CDN

### ❌ Что НЕ используется

- Никаких фреймворков (React, Vue, Bootstrap)
- Никаких сборщиков (Webpack, Vite)
- Никаких плагинов и библиотек

Только **чистый HTML + CSS + JS**.

---

## 📱 Секции лендинга

1. **Header** — sticky, бургер-меню на чистых span
2. **Hero** — главный заголовок, CTA, фоновое изображение
3. **Pain** — блок «Знакомо?» (3 проблемы)
4. **Result** — «После курса вы:» (3 результата)
5. **Program** — аккордеон с 3 модулями и списками уроков
6. **Author** — 3 спикера с фото и статистикой
7. **Tariffs** — 3 тарифа (Базовый / PRO / VIP)
8. **Timer** — обратный отсчёт с пульсирующим свечением
9. **Reviews** — 3 отзыва со звёздами
10. **Contact** — форма заявки с маской телефона
11. **Sticky CTA** — плавающая кнопка снизу
12. **Cookie Consent** — баннер с выбором (Принять / Отклонить)
13. **Policy Modal** — модалка политики конфиденциальности
14. **Footer** — информация о демо-характере, разработчике, Telegram, дата

---

## ✨ Ключевые фичи

- 🍔 Бургер-меню с анимацией span-полосок в крестик (cubic-bezier easing)
- ⏱ Обратный отсчёт с мягким пульсирующим свечением (box-shadow анимация, 2s цикл)
- 📞 Маска телефона `+7 (XXX) XXX-XX-XX` с автоформатированием при вводе
- 🎨 Glassmorphism (`backdrop-filter: blur()`) на хедере и карточках
- ♿ Accessibility: ARIA-атрибуты, роли, `focus-visible`, `prefers-reduced-motion`
- 📄 `@media print` — скрытие лишнего, сброс цветов при печати
- 🍪 Cookie consent с сохранением выбора в `localStorage`
- 🚀 IntersectionObserver вместо событий scroll — производительно
- 📦 IIFE-модули — нет глобальных переменных
